# Code of Conduct

This project is for builders. We keep it simple.

## Our standards

We expect:

- Respectful communication
- Good-faith collaboration
- Clear, constructive feedback
- No harassment, hate speech, or personal attacks
- No doxxing or sharing private information

## Scope

This applies to:

- GitHub issues
- pull requests
- discussions
- any project-related community spaces

## Enforcement

If someone is making the project hostile or unsafe, maintainers may remove content, lock threads, or restrict participation.

Report conduct issues to:

- contact@signallynx.com

If you report an issue, include links and context. We will review and respond.
